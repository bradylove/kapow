h1. KaPow Version .10(alpha3)

!http://content.screencast.com/users/BLove/folders/Jing/media/42075110-953d-4fec-ae88-1e5abc8f82a9/00000008.png!

KaPow is a MacRuby application for managing your symlinks that Pow uses you run your rails apps. "Pow is a zero-config Rack server for Mac OS X. Have it serving your apps locally in under a minute."

Pow must be installed for KaPow to work.

You can install Pow with the following command in your terminal.

<pre>
<code>
$ curl get.pow.cx | sh
</code>
</pre>

Once you have done that you can use KaPow to setup the apps that you would like Pow to serve.

To use KaPow download the latest .zip file from the Downloads section and unzip it. Then just drag KaPow.app to your applications folder and you are good to go.

You can get more information about Pow at http://pow.cx
